# Reminder App

## Description
This project is a reminder application built using Python. It allows users to add, view, and remove reminders with specific dates through a simple menu system.

## Features
- Add reminders with a name and date
- View all reminders in a sorted list
- Remove completed reminders using ID
- Input validation for dates and user input
- Handles invalid inputs and errors

## How to Run
1. Make sure Python is installed
2. Download or clone this repository
3. Open terminal or command prompt
4. Run:
   python reminder.py

## Example
1. Add a reminder  
2. Enter name: Homework  
3. Enter date: 25-12-2025  
4. Reminder is saved and displayed in the list  

## Purpose
The purpose of this project is to practice Python programming skills such as:
- Functions
- Lists and dictionaries
- User input and validation
- Error handling
- Working with dates (datetime module)

## Future Improvements
